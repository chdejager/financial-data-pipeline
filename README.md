# Financial Data Pipeline
A collection of pipelines for discovering financial assets and collecting historical time-series data for quantitative research.

This repository is designed to support research workflows that require structured financial datasets across a large universe of assets.

---
## Repository Structure
'''
finacial-data-pipeline/

finaeon/
  Pipeline for symbol discovery and time-series scraping
'''

Additional data pipelines will be added over time

### Finaeon Pipeline
